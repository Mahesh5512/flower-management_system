# flower-management_system
This project is a full-stack Flower Management System designed to manage flower products, categories, customer details, and orders efficiently. The system provides a simple and user-friendly interface for handling day-to-day operations of a flower shop, with a focus on performance, clarity, and maintainability.

The frontend is built using HTML5 and CSS, ensuring a responsive and clean user experience. The backend is developed using Node.js and Express.js, providing RESTful APIs to handle business logic and data operations. Mysql is used as the primary database to store product, inventory, customer, and order information.

The application supports CRUD operations, search functionality, and basic dashboard insights to help track inventory and orders effectively. This project demonstrates practical full-stack development skills and real-world data handling.

- Features

Product, category, order, and customer management

CRUD operations for all modules

Search and filter functionality

Responsive UI design

Backend REST APIs

Database integration and data persistence

- Tech Stack

Frontend: HTML5, CSS

Backend: Node.js, Express.js

Database: MongoDB

Tools & Other Skills: MySQL, PHP, XAMPP, Python

- Purpose

Built as a hands-on project to strengthen full-stack development skills and demonstrate end-to-end application design.
